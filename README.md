## Proposta

#Este projeto propõe uma aplicação Web para a divulgação e a facilitação de uso do acervo da Biblioteca Municipal Getúlio Vargas. O projeto objetiva a melhoria da divulgação do acervo da biblioteca aos munícipes através do meio digital. Além disto permite que sejam agendados empréstimos de exemplares do catálogo diretamente no site, evitando a necessidade de deslocar-se pessoalmente até a biblioteca para verificar a disponibilidade do(s) item(s) e realizar o empréstimo.
